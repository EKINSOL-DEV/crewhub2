# CrewHub v2

A secure, local-first desktop mission control for Claude Code — built with Tauri v2 (Rust core) and React 19. Ground-up rebuild of [CrewHub](https://github.com/ekinsol/crewhub); v1 remains the OpenClaw build.

- Master plan: [`docs/plans/2026-06-10-crewhub-v2-rebuild.md`](docs/plans/2026-06-10-crewhub-v2-rebuild.md)
